## 🚀 Getting Started

1. **Clone the repo**  
```bash
   git clone https://github.com/GNGscr/IceOfCream.git
   cd IceOfCream
```

2. **Install dependencies**  
```bash  
  npm install
```

3. **Run the dev server**

```bash
    npm run dev
```

The app will run on `http://localhost:3000`.

---

## 📦 Build for production
```bash
  npm run build
```
