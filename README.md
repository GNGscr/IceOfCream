## 🚀 Getting Started

1. **Clone the repo**  
   ```bash
   git clone https://github.com/GNGscr/IceOfCream.git
   cd IceOfCream

2. **Install dependencies**  

  npm install

3. **Start the dev server**

  npm run dev

## 📦 Build for production

  npm run build
